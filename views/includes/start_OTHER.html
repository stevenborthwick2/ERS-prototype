Submit your end of year Other return

Use this service if you operate an Other scheme and need to submit your return at the end of the tax year. This service replaces form 42.

Before you start

You'll need to know if there have been any events in relation to options and/or securities (including shares) in the schemes and arrangements registered under this scheme. If there have been related events, you'll need to record the events in a spreadsheet and save it to your computer in the '.ods' format. Your spreadsheet must have the same name and follow the same structure as the template supplied by HM Revenue and Customs.
